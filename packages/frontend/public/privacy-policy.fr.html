Politique de Confidentialité

Dernière mise à jour : 5 août 2025

Privacy policy - English version

1. Données Collectées

  • Nous collectons des données uniquement pour vous offrir des fonctionnalités
  • Aucune donnée collectée si vous ne créez pas de compte, votre historique est sauvegardé sur votre appareil
  • Si vous créez un compte : nous collectons votre login et votre historique de méditation.
  • Aucun partage de vos données avec des tiers

Créer un compte vous permet de sauvegarder votre historique de méditation sur notre serveur, et ainsi vous pouvez y accéder à partir de plusieurs appareils et de le récupérer si vous perdez votre appareil, si vous désinstallez ou supprimez les données de l'application et si vous supprimez les données de votre navigateur web

2. Sécurité

Vos données sont stockées de manière sécurisée (protocoles HTTPS, base de données sécurisées, hégergeur : https://www.koyeb.com).

Aucun mot de passe n'est demandé, un utilisateur connaissant votre login peut donc avoir accès à votre historique de méditation. Cependant, nous avons jugé que ces données n'étaient pas critiques et que la simplicité d'utilisation l'emporte sur le risque. Si vous souhaitez un compte plus sécurisé, contactez-nous (cf. ci-dessous)

3. Vos Droits

Selon le Règlement Général de Protection des Données (RGPD), vous pouvez à tout moment :

  • Supprimer votre compte (et toutes vos données) en nous le demandant
  • Demander une copie de vos données

4. Cookies

  • Aucun cookie utilisé
  • IndexDb est utilisé à la place des cookies pour sauvegarder les données sur votre appareil et pouvoir utiliser l'application hors ligne

5. Modifications

Cette politique peut être mise à jour. Vous serez prévenu(e) dans l'application.
